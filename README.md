Command line interface for monitoring a distributed management system (cli-mon-rsu)
===================================================================================

Introduction
------------
This application is designed as a command line interface (CLI) based on PHP. It is designed to receive data from the Foxboro management system (I/A Series), save the received data to a database (MySQL or SQLite) and CSV files. The application is made according to the software template Model-View-Control (MVC). The data was obtained from the history of the control system using the ODBC driver. Designed to run on Windows.

Installation
------------
Clone or download [cli-mon-rsu](https://github.com/bsa-git/cli-mon-rsu) project with git.
