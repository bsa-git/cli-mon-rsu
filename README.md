Command line interface for monitoring a distributed management system (cli-mon-rsu)
===================================================================================

Introduction
------------
This application is designed as a command line interface (CLI) based on PHP. It is designed to receive data from the Foxboro management system, save the received data to a database and CSV files. Designed to run on Windows.

Installation
------------
Clone or download [cli-mon-rsu](https://github.com/bsa-git/cli-mon-rsu) project with git.

Description
-----------
The application is made according to the software template Model-View-Control (MVC). The data was obtained from the history of the control system using the ODBC driver.
